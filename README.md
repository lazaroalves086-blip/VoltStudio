## 🛠️ Sobre a Estrutura deste Site
Este projeto foi desenvolvido com foco em **velocidade de carregamento** e **conversão de vendas**, utilizando uma arquitetura 100% estática e limpa para garantir que nossos clientes tenham uma navegação fluida em qualquer dispositivo.

* **HTML5:** Semântica estruturada.
* **CSS3:** Identidade visual e responsividade, utilizando variáveis nativas para manter o *Dark Mode* profundo da nossa marca.
* **JavaScript (Vanilla):** Motor de partículas interativas do nosso background em tempo real, sem uso de bibliotecas pesadas.
